Introduction

JavaScript is a versatile programming language primarily used in web development.

It enables interactive behavior on web pages and supports functional, event-driven, and object-oriented programming styles.

console.log("Hello, world!");

JavaScript runs in the browser and can interact with HTML and CSS.

  • Client-side scripting
  • Lightweight and fast
  • Interpreted language
Variables

Variables store data that can be used and modified later.

Use var, let, or const to declare variables.

let name = "Alice";

const is used when the value should not change.

  • let for block-scoped variables
  • const for constants
  • var is function-scoped (older usage)
Functions

Functions allow you to encapsulate reusable blocks of code.

You can define functions using the function keyword or as arrow functions.

function greet(name) { return "Hello " + name; }

Arrow functions are shorter: const greet = name => "Hello " + name;

  • Reusable logic
  • Can return values
  • Can take parameters
Loops

Loops are used to repeat actions multiple times.

Common loop types are for, while, and do...while.

for(let i = 0; i < 5; i++) { console.log(i); }

Loops can iterate over arrays and objects.

  • for loop
  • while loop
  • do...while loop
Arrays

Arrays hold ordered collections of data.

Access elements using index notation.

let fruits = ["apple", "banana", "cherry"];

You can add or remove items using array methods.

  • push()
  • pop()
  • map(), filter(), reduce()
